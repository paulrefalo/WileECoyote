# WileECoyote
Making a prototype app for iOS iPhone to explore parts diagrams, add parts to a shopping cart,
and view orders on both a queue and main dashboard

## Getting Started

Explore with Xcode or install the app on an iOS device

### Prerequisites

An iOS device capable of running Version 8.3.3 (8E3004b) 
```
An iOS device capable of running Version 8.3.3 (8E3004b) 
Developed with Swift 3.1
```
## Built With

* XCode Version 8.3.3 (8E3004b) 
* Developed with Swift 3.1
